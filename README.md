# AI Web Chat Application

## Installation & Running Locally

### Prerequisites
- Node.js installed on your machine
- A valid Groq API Key

### Setup Instructions

1. **Install Dependencies**:
   Open your terminal or command prompt in the root directory of the project and run:
   ```bash
   npm install
   ```

2. **Configure Environment Variables**:
   Create a `.env` file in the same directory as `server.js` (if it doesn't already exist). Insert your API key like this:
   ```env
   GROQ_API_KEY=your_api_key_here
   ```

3. **Run the Application**:
   Start the Node.js server by running:
   ```bash
   npm start
   ```

4. **View the App**:
   Open your web browser and navigate to exactly: `http://localhost:3000`
